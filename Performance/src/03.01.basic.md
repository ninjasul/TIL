	2. HTTP의 이해
		2.1 프로토콜구조
			2.1.1 특징
				- 서버간 요청과 응답이 1:1, 즉, 화면 구성 컨텐츠가 100개라면 요청과 응답이 100번 이루어짐.
			
			2.1.2 헤더
				- 처리 대상, 결과 코드, 데이터 처리를 위한 각종 속성 정보 포함		
				- GET 방식의 경우 본문이 존재하지 않음.
				- 헤더 속성 간 구분자는 [CLRF]임. 16진수로는 [0D0A]
				
			2.1.3 본문
				- 헤더와 본문의 구분자는 [CLRFCLRF]임. 16진수로는 [0D0A0D0A]
